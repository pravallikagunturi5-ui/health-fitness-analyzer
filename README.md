🩺 Health & Fitness Analyzer

A modern **Health & Fitness Analyzer** developed using **Python Flask** that helps users evaluate their overall health based on Body Mass Index (BMI), Basal Metabolic Rate (BMR), ideal weight range, and daily water intake. The application provides personalized health recommendations through an attractive and responsive web interface.


📖 Project Overview

The Health & Fitness Analyzer is designed to help users understand their physical health by entering basic details such as age, gender, height, and weight. The application performs various health-related calculations and displays personalized recommendations in an interactive dashboard.
This project demonstrates the use of **Python Flask**, **HTML**, **CSS**, and **Jinja2 Templates** to build a responsive web application.

✨ Features

- 🔹 Body Mass Index (BMI) Calculation
- 🔹 Gender-Based Daily Calorie Requirement (BMR)

Installation Guide

### 1. Clone the Repository

```bash
git clone https://github.com/PriyaGunturi/health-fitness-analyzer.git
```

### 2. Open the Project Folder

```bash
cd "BMI CALCULATOR"
```

### 3. Install Required Packages

```bash
pip install flask
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open in Browser

```
http://127.0.0.1:5000
```
